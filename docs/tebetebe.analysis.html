get_routes(origins, dests, origin_id_col=None, dest_id_col=None)[source]¶

Get a GeoDataFrame of routes between each origin and destination. This is probably not the most efficient way to do this! Both datasets should be in EPSG:4326

Parameters

  • origins (POIDataset) – Origin points to be used in routing

  • dests (POIDataset) – Destination points to be used in routing

  • origin_id_col (str, optional) – Column in origins dataset to be used as origin_id row in returned GDF. ID col must be unique if specified; if not, the row index will be used

  • dest_id_col (str, optional) – Column in dests dataset to be used as dest_id row in returned GDF. ID col must be unique if specified; if not, the row index will be used

Returns

Data Frame with “origin_id”, “dest_id”, “time”, “dist” columns, and a geometry column with the route calculated by OSRM.

Return type

gpd.GeoDataFrame

time_matrix(origins, dests, origin_id_col=None, dest_id_col=None)[source]¶

Return a time matrix of origins to destinations, in a DataFrame.

Parameters

  • origins (POIDataset) – Origin points to be used in time matrix (these will be returned as rows)

  • dests (POIDataset) – Destination points to be used in time matrix (these will be returned as columns)

  • origin_id_col (str, optional) – Column in origins dataset to be used as index for time matrix. ID col must be unique if specified; if not, the row index will be used

  • dest_id_col (str, optional) – Column in dests dataset to be used as index for time matrix. ID col must be unique if specified; if not, the row index will be used

Returns

Matrix dataframe of time, in seconds, from each origin to each destination.

Return type

pd.GeoDataFrame
